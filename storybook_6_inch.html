286×146mm 横版双图画布

300dpi，3mm 出血线。图片左右对称；可导出最高分辨率 PNG/ZIP/PDF。

当前画布列表固定在这里。可拖拽排序，ZIP 按此顺序导出。
背景与辅助线
背景编辑默认锁定。点击“背景编辑：解锁”后，才可在预览区空白处拖动背景或滚轮缩放。
双正方形图片
左右文本框
文本框与对应图框中心对齐；文字顶部对齐。预览中可拖左上角移动，拖右下角调宽高。
字体与样式
默认不自动加载 GitHub 字体库，避免大量字体占用启动网络和内存。需要时点击上方按钮手动加载；现在读取 my-fonts-lib/chinese 和 my-fonts-lib/english，优先通过 GitHub Pages 访问，失败时自动尝试 raw.githubusercontent.com。
设置导出 / 导入
设置 JSON 可保存所有可调参数，也可保存图片文件名/相对路径。浏览器不能静默读取本地路径；导入设置后，请分别选择“图片文件夹”和“背景文件夹”，系统会按相对路径、文件名和大小匹配左/右图与背景。多选图片补充加载可作为兜底。
点击画布空白处改名；上传背景后可拖动/滚轮缩放背景；点击图框上传。导出为 286×146mm / 300dpi，PDF 为精确页面尺寸。
